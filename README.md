## Project Overview

This repository contains a detailed optimization framework for a portfolio of 5 Airbnb properties:
- **4 Hartford, CT apartments** (Studio, 1BR, 2BR, 3BR)
- **1 Sarasota, FL vacation home** (4BR with heated pool)

The analysis covers two distinct markets with different demand patterns, seasonality, and target audiences.

## Skills Demonstrated

- **Market Research & Analysis** - Competitor benchmarking, market sizing, demand pattern identification
- **SEO Optimization** - Title optimization, keyword research, description copywriting
- **Data Analysis** - Occupancy trends, conversion metrics, pricing analysis
- **Strategic Planning** - SWOT analysis, guest persona development, go-to-market strategy
- **Content Strategy** - Photo ordering optimization, caption writing, listing copy

## Key Findings

### Hartford Market
| Metric | Value |
|--------|-------|

| Average Daily Rate | $102 |

### Sarasota Market
| Metric | Value |
|--------|-------|

## Key Recommendations Summary

### Immediate Actions
- Implement dynamic pricing for seasonal adjustments
- Optimize listing titles with high-value keywords
- Reorder photos with hero shots first
- Enable weekly/monthly discounts for extended stays

### Strategic Initiatives
- Develop travel nurse marketing program
- Create snowbird return-guest loyalty program
- Expand to multi-platform distribution (VRBO, Furnished Finder)
- Build corporate housing partnerships
